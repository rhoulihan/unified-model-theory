The UMT Framework

Model Once. Project for Every Consumer.

JSON Relational Duality in Action

CREATE JSON RELATIONAL DUALITY VIEW order_v AS
SELECT JSON {
  '_id': o.order_id,
  'customer': {
    'name': c.name,
    'email': c.email
  },
  'items': [
    SELECT {'sku': li.sku, 'qty': li.quantity}
    FROM line_items li
    WHERE li.order_id = o.id
  ]
}
FROM orders o
JOIN customers c ON o.customer_id = c.id;

One declaration. Bidirectional. Full ACID. All three dimensions.
